1 Swap values

a, b = b, a
Lesson: Lua parallel assignment → no temp var ever needed.

2 Ternary via and/or

local x = (n > 0) and "pos" or "neg"
Lesson: Breaks if true-branch = false/nil. Know the trap before using.

3 Default args

function f(x) x = x or 42; return x end
Lesson: or = default. Fails when false is a valid input.

4 Push → return item

function push(t, x) t[1+#t] = x; return x end
Lesson: Returning pushed item → chainable ops. 1+#t faster than table.insert.

5 Map (non-destructive)

function map(t, f,    u)
  u = {}; for _, x in ipairs(t) do u[1+#u] = f(x) end; return u end
-- map → if f returns nil, item skipped (u[1+#u]=nil is no-op)
-- select:  map(t, function(x) if x > 0 then return x end end)
-- reject:  map(t, function(x) if x <= 0 then return x end end)
Lesson: map = map + select + reject. Return nil → skip. One fn, three uses. No separate filter needed.

6 Chainable sort

function sort(t, fn) table.sort(t, fn); return t end
Lesson: table.sort returns nil. Wrapping → chainable: sort(map(t,f)).

7 Sum with function

function sum(t, f,    n)
  n = 0; for _, v in pairs(t) do n = n + (f and f(v) or v) end; return n end
-- uses pairs not ipairs, so works on dicts too
-- sum({1,2,3})          --> 6
-- sum({1,2,3}, sq)      --> 14
-- sum(map(t, f))        --> map then sum
Lesson: sum = reduce(+). With fn arg → no intermediate table. Without fn → sum of map result. Compose: sum(map(t,f)).

8 Slice

function slice(t, lo, hi,    u)
  lo = lo and (lo < 0 and #t+1+lo or lo) or 1
  hi = hi and (hi < 0 and #t+1+hi or hi) or #t
  u = {}; for i = lo, hi do u[1+#u] = t[i] end; return u end
-- slice({10,20,30,40,50}, -3)    --> {30,40,50}
-- slice({10,20,30,40,50}, 2, -2) --> {20,30,40}
Lesson: Negative index = count from end. -1 = last, -2 = second-to-last. Same convention as Python. Lua has no built-in slice — write it once, use forever.

9 Fisher-Yates shuffle

function shuffle(t,    j)
  for i = #t, 2, -1 do j = math.random(i); t[i], t[j] = t[j], t[i] end
  return t end
Lesson: Only correct O(n) shuffle. Don’t sort with random — biased distribution.

10 Random sample

function many(t, n) return slice(shuffle(t), 1, n) end
Lesson: Compose primitives. shuffle+slice = reservoir sampling without complexity.

11 OO in 2 lines

function new(kl, obj)
  kl.__index = kl; return setmetatable(obj, kl) end
Lesson: Entire class system = metatable + __index. No framework needed.

12 Polymorphic dispatch

local NUM, SYM = {}, {}
function NUM.mid(i) return i.mu end
function SYM.mid(i) return i.mode end
-- col:mid() dispatches based on metatable
Lesson: Metatable __index → automatic dispatch. No if type== chains.

13 CSV iterator

function csv(file,    src)
  src = io.open(file)
  return function(    s, t)
    s = src:read()
    if s then
      t = {}; for x in s:gmatch"[^,]+" do t[1+#t] = x end; return t
    else src:close() end end end
Lesson: Iterators = lazy. 1M row file → constant memory.

14 Coerce string → typed value

function thing(s)
  return s == "true" or (s ~= "false" and (tonumber(s) or s)) end
Lesson: One line replaces JSON parser for simple configs. "true"→true, "42"→42, "cat"→"cat".

15 CLI from help string

for k, v in help:gmatch("(%S+)=(%S+)") do the[k] = thing(v) end
Lesson: Help string IS config spec. Single source of truth. DRY to the bone.

16 Trim whitespace

function trim(s) return s:match"^%s*(.-)%s*$" end
Lesson: .- = non-greedy. %s*(.-)%s*$ trims both sides.

17 Pretty-print anything

function o(x,    u)
  if type(x) ~= "table" then
    return math.type(x) == "float" and string.format("%.2f", x)
           or tostring(x) end
  u = {}
  for k, v in pairs(x) do
    u[1+#u] = type(k) == "number" and o(v) or k .. "=" .. o(v) end
  return "{" .. table.concat(#x==0 and sort(u) or u, ", ") .. "}" end
Lesson: Recursive o() → debug anything. print(o(x)) replaces entire inspect library.

18 Dot product

function dot(a, b,    n)
  n = 0; for i = 1, #a do n = n + a[i] * b[i] end; return n end
Lesson: No zip needed. Index loop = fastest path in Lua.

19 Running mean (Welford)

i.n = i.n + 1
local d = v - i.mu
i.mu = i.mu + d / i.n
Lesson: One-pass mean. No need to store all values. Numerically stable.

20 Running variance (Welford)

i.n  = i.n + 1
local d = v - i.mu
i.mu = i.mu + d / i.n
i.m2 = i.m2 + d * (v - i.mu)
sd = i.n < 2 and 0 or math.sqrt(math.max(0,i.m2) / (i.n - 1))
Lesson: Online variance in 4 lines. m2 tracks sum of squared diffs from running mean. Guard n<2 and max(0,m2) to avoid NaN from floating point drift.

21 Entropy

function entropy(has, n,    f)
  f = function(v) return v > 0 and -(v/n)*math.log(v/n,2) or 0 end
  return sum(has, f) end
-- entropy({a=3, b=1, c=1}, 5) --> 1.37
Lesson: Entropy defined via sum. Compose primitives → no manual loops. Low entropy = pure, high = mixed.

22 Mode (most common)

function mode(has) return max(has) end
-- mode({a=3, b=1, c=1}) --> "a"
Lesson: Mode = key with max count. One call to max. Compose primitives → no custom loops.

23 Sigmoid normalization

function sigmoid(z) return 1 / (1 + math.exp(-1.7 * z)) end
Lesson: Maps (−∞, ∞) → (0, 1). The 1.7 scaling ≈ matches CDF of normal distribution.

24 Binary search (bisect)

function bisect(t, x,    lo, hi, m)
  lo, hi = 1, #t
  while lo <= hi do
    m = (lo + hi) // 2
    if t[m] <= x then lo = m + 1 else hi = m - 1 end end
  return lo - 1 end
Lesson: Returns insertion point. Foundation for rank stats, KS test, percentiles.

25 Weibull random variate

function weibull(k, lambda)
  return lambda * (-math.log(1 - math.random()))^(1/k) end
Lesson: Inverse transform sampling. Works for any distribution with closed-form inverse CDF.

26 FizzBuzz

for i = 1, 100 do
  print((i%15==0 and "FizzBuzz") or (i%3==0 and "Fizz")
        or (i%5==0 and "Buzz") or i) end
Lesson: Chain and/or = multi-branch ternary. Check most specific first.

27 Flatten nested table

function flat(t, u)
  u = u or {}
  for _, v in ipairs(t) do
    if type(v) == "table" then flat(v, u) else u[1+#u] = v end end
  return u end
Lesson: Recursive + accumulator arg → no intermediate tables.

28 Key-value transform

function kv(t, fk, fv,    u)
  u = {}; for _, x in ipairs(t) do u[fk(x)] = fv(x) end; return u end
Lesson: Build dicts from lists in one pass. Replaces 5-line loop patterns.

29 Select / reject (via map)

function sel(t, f) return map(t, function(x) if     f(x) then return x end end) end
function rej(t, f) return map(t, function(x) if not f(x) then return x end end) end
-- sel({1,-2,3,-4}, function(x) return x>0 end) --> {1,3}
-- rej({1,-2,3,-4}, function(x) return x>0 end) --> {-2,-4}
Lesson: sel and rej = thin wrappers on map. No new loops. Compose, don't repeat.

30 Max (key of highest value)

function max(t, f,    best, bv)
  for k, v in pairs(t) do
    local s = f and f(k, v) or v
    if not bv or s > bv then best, bv = k, s end end
  return best, bv end
-- max({a=3, b=1})                  --> "a", 3
-- max(t, function(k,v) return v end) --> argmax with custom fn
Lesson: Without fn → key of max value (used by mode). With fn → generic argmax. One fn, two uses.

31 Reverse table

function rev(t,    u)
  u = {}; for i = #t, 1, -1 do u[1+#u] = t[i] end; return u end
Lesson: New table, not in-place. Immutable style = safe to pass around.

32 Unique values (order preserved)

function unique(t,    seen, u)
  seen, u = {}, {}
  for _, v in ipairs(t) do
    if not seen[v] then seen[v] = true; u[1+#u] = v end end
  return u end
Lesson: Hash table for O(1) lookup. Preserves insertion order unlike set-by-keys.

33 Prefix sum / accumulate

function accum(t,    u, s)
  u, s = {}, 0
  for _, v in ipairs(t) do s = s + v; u[1+#u] = s end
  return u end
Lesson: Running totals. Foundation for CDF, percentile calculations.

34 Quicksort (functional)

function qsort(t)
  if #t <= 1 then return t end
  local p, lo, hi = t[1], {}, {}
  for i = 2, #t do push(t[i] < p and lo or hi, t[i]) end
  return flat{qsort(lo), {p}, qsort(hi)} end
Lesson: Elegant but slow — O(n) extra alloc per level. Use table.sort in prod.

35 Memoize

function memo(f,    cache)
  cache = {}
  return function(x) cache[x] = cache[x] or f(x); return cache[x] end end
Lesson: Trade memory for speed. Only works for pure fns with hashable args.

36 Coroutine iterator

function range(lo, hi)
  return coroutine.wrap(function()
    for i = lo, hi do coroutine.yield(i) end
  end) end
Lesson: Coroutines = custom iterators. Lazy, composable, zero-copy.

37 Pipe / compose

function pipe(x, ...)
  for _, f in ipairs{...} do x = f(x) end; return x end
-- pipe(data, clean, normalize, classify)
Lesson: Left-to-right composition. Reads like English. No nesting.

38 Crop / clamp

function crop(x, lo, hi) return math.max(lo, math.min(hi, x)) end
Lesson: Clamp in one line. Use everywhere: pixel coords, probability, config bounds.

39 Naive Bayes likelihood

function like(has, v, n, prior, k)
  return math.max(1/1E32, ((has[v] or 0) + k*prior) / (n + k + 1/1E32)) end
Lesson: Laplace smoothing (k*prior) prevents zero-probability trap. Never multiply by 0.

40 Log-sum for Bayes

score = math.log(prior) + sum(cols, function(c)
  return c.v ~= "?" and math.log(c:like(v)) or 0 end)
Lesson: Log-space → additions not multiplications. No float underflow on many features.

41 Even check (bitwise)

local is_even = (n & 1) == 0
Lesson: Bitwise AND → fastest parity check. No modulo needed.

42 String interpolation

function fmt(s, t) return s:gsub("{(%w+)}", t) end
-- fmt("Hello {name}!", {name = "world"})
Lesson: gsub + table = template engine in 1 line.

43 Deep copy

function copy(t)
  if type(t) ~= "table" then return t end
  local u = {}
  for k, v in pairs(t) do u[k] = copy(v) end
  return setmetatable(u, getmetatable(t)) end
Lesson: Preserves metatables. Recursive. Needed when modifying cloned configs.

44 Group by key function

function group(t, f,    u)
  u = {}
  for _, x in ipairs(t) do
    local k = f(x); u[k] = u[k] or {}; u[k][1+#u[k]] = x end
  return u end
Lesson: SQL GROUP BY in pure Lua. Foundation for stratified sampling, pivots.

45 Zip two lists

function zip(a, b,    u)
  u = {}; for i = 1, math.min(#a, #b) do u[i] = {a[i], b[i]} end
  return u end
Lesson: No built-in zip. 1 line. Use math.min to handle unequal lengths.

46 Dict from key/val lists

function dict(ks, vs,    u)
  u = {}; for i = 1, #ks do u[ks[i]] = vs[i] end; return u end
Lesson: Python’s dict(zip(k,v)) in Lua. Pairs two parallel arrays.

47 Count occurrences

function counts(t,    u)
  u = {}; for _, v in ipairs(t) do u[v] = (u[v] or 0) + 1 end; return u end
Lesson: Histogram in one pass. Foundation for entropy, mode, chi-square.

48 Curry (partial application)

function curry(f, x) return function(...) return f(x, ...) end end
Lesson: Lock first arg → specialized fn. curry(add, 1) = increment.

49 CLI dispatcher

-- Config from help string (help = "... seed=1 ... k=2 ...")
for k, v in help:gmatch("(%S+)=(%S+)") do the[k] = thing(v) end

-- CLI: flags update config or call demo functions
local function main(    n, k, v)
  n = 1
  while n <= #arg do
    n, k, v = n+1, arg[n], arg[n+1]
    if eg[k] then math.randomseed(the.seed); eg[k](v and thing(v) or nil)
              if v and not eg[v] then n = n+1 end
    else for k1 in pairs(the) do
           if k == "-"..k1:sub(1,1) then the[k1] = thing(v); n = n+1 end end end end end

-- lua ezr.lua -s 42 --test data.csv
Lesson: Help string = config spec = docs. CLI parses flags into the or dispatches to eg table. Zero dependencies. Whole CLI in 10 lines.

50 Test harness in 10 lines

local eg = {}

eg["--all"] = function(_,    ss)
  ss = {}
  for k in pairs(eg) do if k ~= "--all" then ss[1+#ss] = k end end
  table.sort(ss)
  for _, k in ipairs(ss) do
    print("\n" .. k); math.randomseed(1); eg[k]() end end
Lesson: Tests = table of fns. --all = iterate sorted keys. Seed reset → reproducible. No test framework needed.

TL;DR

Lua rewards small composable functions. Build push, map, sort, sum, sel once → everything else is 1–3 lines. Help string as config. Metatables as classes. Iterators as lazy streams. Keep stdlib tiny, compose big.
